📖 Ramadan Quran Reader
🌙 About the Project

This is a Ramadan Quran reading assistant web app built using Python and Streamlit.

The application helps users:

📖 Read Quran pages sequentially

📅 Schedule Quran ختم over chosen days

⭐ Play a simple Tasbih 

🤲 Display daily duaa

🚀 Technologies Used

Python

Streamlit

PyMuPDF (fitz)

Pillow
